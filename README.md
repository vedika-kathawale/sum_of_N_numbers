# sum_of_N_numbers
A simple, interactive Sum of N Numbers web application that allows users to calculate the sum of any number of inputs dynamically. This project focuses on clean UI design, smooth user interaction, and beginner-friendly JavaScript logic.
Users first enter how many numbers (N) they want to add, generate the required input fields automatically, and then calculate the total sum with a single click.

# Key Features

1.  Dynamically generate input fields based on user-defined N
2.  Calculate the total sum instantly
3.  Input validation for non-numeric or empty values
4.  Clean, modern UI with soft colors and rounded elements
5.  Fully responsive design
6.  Smooth user experience using JavaScript DOM manipulation
